🌾 Agricultural Data Visualization & Insights (Power BI Dashboard)
📌 Project Overview

This project presents an interactive Power BI dashboard that analyzes agricultural production data across different states, crops, and seasons in India.
The goal is to provide data-driven insights for better understanding of crop production trends, seasonal impact, and regional performance.

🎯 Objectives
Analyze total agricultural production
Identify top-performing crops
Compare state-wise production
Study season-wise distribution
Observe yearly production trends
Highlight top contributing crops
📊 Dashboard Features
